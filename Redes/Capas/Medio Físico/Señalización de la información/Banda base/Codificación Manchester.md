La codificación Manchester emplea transiciones de nivel. Esto permite:
- Sincronización emisor-receptor más fácil -> señal de reloj intrínseca 
Si pasa de 0 a 1 -> valor lógico = 1
Si pasa de 1 a 0 -> valor lógico = 0
